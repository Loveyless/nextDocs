---
title: Next.js 编译器
description: Next.js 编译器，用 Rust 编写，可转换和压缩你的 Next.js 应用。
---

Next.js 编译器使用 [SWC](https://swc.rs/) 用 Rust 编写，允许 Next.js 转换和压缩你的 JavaScript 代码以供生产使用。这取代了用于单个文件的 Babel 和用于压缩输出包的 Terser。

使用 Next.js 编译器进行编译比 Babel 快 17 倍，自 Next.js 版本 12 起默认启用。如果你现有的 Babel 配置或正在使用[不支持的功能](#unsupported-features)，你的应用将选择退出 Next.js 编译器并继续使用 Babel。

欢迎使用 Next.js 文档！

## 什么是 Next.js？

Next.js 是一个用于构建全栈 Web 应用的 React 框架。你可以使用 React 组件构建用户界面，并使用 Next.js 提供的额外功能和优化。

它还会自动配置底层工具（如打包器和编译器），让你可以专注于构建产品和快速发布。

无论你是个人开发者还是大型团队的一员，Next.js 都能帮助你构建交互式、动态且快速的 React 应用。

## App Router 和 Pages Router

Next.js 有两种不同的路由：

- **App Router**：较新的路由，支持 Server Components 等新 React 特性。
- **Pages Router**：原始路由，仍在支持并持续改进。

在侧边栏顶部，你会看到一个下拉菜单，允许你在 [App Router](/docs/app) 和 [Pages Router](/docs/pages) 文档之间切换。

### React 版本处理

App Router 和 Pages Router 处理 React 版本的方式不同：

- **App Router**：内置使用 [React canary releases](https://react.dev/blog/2023/05/03/react-canaries)，包含所有稳定的 React 19 变更，以及在新 React 发布前正在框架中验证的新功能。

- **Pages Router**：使用项目中 `package.json` 安装的 React 版本。

这种方法既确保了新 React 功能在 App Router 中可靠运行，又保持了与现有 Pages Router 应用的向后兼容性。

> **了解更多**：`getInitialProps` 是一个旧版 API。我们建议使用 [`getStaticProps`](/docs/pages/building-your-application/data-fetching/get-static-props) 或 [`getServerSideProps`](/docs/pages/building-your-application/data-fetching/get-server-side-props) 代替。

`getInitialProps` 是一个`异步`函数，可以添加到页面的默认导出 React 组件中。它将在服务器端运行，并在页面转换期间再次在客户端运行。函数的结果将作为 `props` 转发到 React 组件。

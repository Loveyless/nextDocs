---
title: Image (旧版)
description: 使用旧版 Image 组件向后兼容图像优化。
version: legacy
---

从 Next.js 13 开始，`next/image` 组件被重写以提高性能和开发者体验。为了提供向后兼容的升级解决方案，旧的 `next/image` 被重命名为 `next/legacy/image`。

> **警告**：`next/legacy/image` 已被弃用，将在未来版本的 Next.js 中移除。请改用 [`next/image`](/docs/app/api-reference/components/image)。

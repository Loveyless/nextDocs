---
title: 客户端获取
description: 了解客户端数据获取，以及如何使用 SWR，一个处理缓存、重新验证、焦点跟踪、间隔重新获取等的数据获取 React Hook 库。
---

当你的页面不需要 SEO 索引、不需要预渲染数据，或者页面内容需要频繁更新时，客户端数据获取很有用。与服务器端渲染 API 不同，你可以在组件级别使用客户端数据获取。

如果在页面级别完成，数据在运行时获取，页面内容随着数据变化而更新。在组件级别使用时，数据在组件挂载时获取，组件内容随着数据变化而更新。

重要的是要注意，使用客户端数据获取会影响你的应用性能和页面加载速度。这是因为数据获取是在组件或页面挂载时完成的，并且数据没有被缓存。
